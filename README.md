# Real Estate Analytics Dashboard


This project is a comprehensive **real estate analytics dashboard** built in Microsoft Excel. It leverages raw transaction data to generate insights into property sales, agent performance, regional trends, and year‑over‑year growth. The dashboard demonstrates skills in **data cleaning, PivotTables, PivotCharts, and KPI visualization**.


- **Raw Data Integration**: Transaction records including property type, location, bedrooms, bathrooms, floor area, lot area, listing price, sale price, agent, buyer type, payment method, and commission.
- **PivotTable Summaries**:
  - Sales by Property Type (Commercial, Condo, House & Lot, Land).
  - Sales by Location (Bulacan, Cavite, Laguna, Makati).
  - Top Performing Agents (commissions and total sales).
  - Year‑over‑Year Growth in sales revenue.
  - Monthly Sales Trends across multiple years.
- **Interactive Dashboard**:
  - KPI cards for total sales, commissions, and growth.
  - Charts for property type distribution, location breakdown, agent performance, and monthly trends.
  - Slicers for filtering by year, month, property type, and location.
  - 
1. Raw transaction data is stored in the **Raw_Data** sheet.
2. PivotTables summarize sales by property type, location, agent, and time period.
3. PivotCharts visualize these summaries for quick insights.
4. KPI cards highlight key metrics such as total sales, commissions, and growth percentages.
5. Slicers allow interactive filtering across dimensions (year, property type, city).

- Excel Tables for structured data.
- PivotTables & PivotCharts for analytics.
- Conditional formatting for KPI cards.
- Interactive slicers for user‑friendly filtering.
